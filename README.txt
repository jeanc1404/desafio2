Siguiendo la idea de mi proyecto final de un simulador de compra de boletos aereos, acá una 
pequeña función para el calculo de un descuento según la edad del pasajero. Solicitando al usuario ingresar la edad
seguido devuelve el monto de descuento y el total. Las condiciones para aplicar descuentos son: pasajeros menores de 12 años
o mayores de 65 años.
